SCSServiceConfiguration

@interface SCSServiceConfiguration : NSObject

Class used to specify Service Cloud configuration settings.

Pass an instance of this class to the SCServiceCloud.serviceConfiguration property on the SCServiceCloud shared instance to set the configuration.
